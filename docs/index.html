AI System Engineer Portfolio

Production-ready AI infrastructure portfolio focused on GPU-based AI serving, MLOps, air-gapped deployment, and reliability engineering.

Projects

Project 1 - GPU-Based AI Service Infrastructure (On-Prem)

Built an on-premise AI inference platform using Ubuntu, Docker, NVIDIA GPU runtime, FastAPI, Hugging Face Transformers, and Nginx reverse proxy.

Key features:

  • GPU-enabled Docker runtime
  • FastAPI inference API
  • Single and batch prediction endpoints
  • Request logging and error handling
  • Nginx reverse proxy
  • CPU fallback mode for runtime stability

Project 2 - MLOps End-to-End Pipeline

Project 3 - Air-Gapped AI System

Project 4 - Reliability Engineering
